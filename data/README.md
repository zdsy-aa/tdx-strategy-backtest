# 数据目录 (data/)

本目录存放项目所需的所有股票数据和回测结果。

## 目录结构

```
data/
├── day/                   # 日线数据目录
│   ├── sh/                # 上海证券交易所 (6开头, sh开头)
│   ├── sz/                # 深圳证券交易所 (0/3开头, sz开头)
│   ├── bj/                # 北京证券交易所 (4/8开头, bj开头)
│   └── other/             # 其他无法分类的数据
│
└── backtest_results/      # 回测结果目录
    ├── backtest_results.json  # 策略回测汇总数据
    └── stock_reports.json     # 单只股票详细回测数据
```

## 数据格式说明

所有日线数据均为 CSV 格式，包含以下列：
*   `日期`: 交易日期 (YYYY-MM-DD)
*   `开盘`: 开盘价
*   `收盘`: 收盘价
*   `最高`: 最高价
*   `最低`: 最低价
*   `成交量`: 成交数量

## 数据获取

请运行 `py_file/download_full_market.py` 脚本自动下载和更新数据。该脚本会自动将下载的数据按市场分类存放到对应的子目录中。
