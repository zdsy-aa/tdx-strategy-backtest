# 回测报告目录 (report/)

> 存储所有回测分析报告，按总体、年度、月度三个维度分类。

## 📁 目录结构

```
report/
├── README.md                # 本说明文件
├── summary/                 # 总体报告
│   ├── overall_summary.md   # 全局汇总报告
│   ├── six_veins_report.md  # 六脉神剑专项报告
│   ├── combo_report.md      # 组合策略报告
│   └── best_strategies.md   # 最优策略汇总
├── yearly/                  # 年度报告
│   ├── 2024_report.md       # 2024年度报告
│   ├── 2023_report.md       # 2023年度报告
│   ├── 2022_report.md       # 2022年度报告
│   └── ...
└── monthly/                 # 月度报告
    ├── 2024_12_report.md    # 2024年12月报告
    ├── 2024_11_report.md    # 2024年11月报告
    └── ...
```

## 📊 报告类型说明

### 1. 总体报告 (summary/)

汇总整个回测周期的核心数据：

| 报告文件 | 内容 |
|---------|------|
| `overall_summary.md` | 所有策略的综合对比 |
| `six_veins_report.md` | 六脉神剑各组合详细分析 |
| `combo_report.md` | 组合策略回测结果 |
| `best_strategies.md` | 最优策略推荐 |

### 2. 年度报告 (yearly/)

按年度统计的策略表现：

- 年度胜率统计
- 年度收益曲线
- 年度最优策略
- 与大盘对比分析

### 3. 月度报告 (monthly/)

按月度统计的策略表现：

- 月度信号触发次数
- 月度胜率变化
- 月度收益分布
- 市场环境分析

## 📝 报告格式

所有报告采用 Markdown 格式，包含以下标准结构：

```markdown
# 报告标题

## 概述
- 报告周期
- 数据来源
- 测试标的

## 核心数据
| 指标 | 数值 |
|------|------|
| 胜率 | XX% |
| 平均收益 | X.XX% |
| 最大回撤 | X.XX% |

## 详细分析
...

## 结论与建议
...
```

## 🔄 报告生成

### 自动生成
运行回测脚本时会自动生成对应报告：

```bash
# 生成六脉神剑报告
cd py_file
python 101_six_veins_test.py
# 报告输出到 report/summary/six_veins_report.md

# 生成组合策略报告
python 201_steady_combo_test.py
# 报告输出到 report/summary/combo_report.md
```

### 手动生成
```bash
# 生成年度报告
python generate_yearly_report.py --year 2024

# 生成月度报告
python generate_monthly_report.py --year 2024 --month 12
```

## 🌐 网站查看

报告可通过网站的「回测报告」页面在线查看：

1. 访问网站首页
2. 点击导航栏「工具」→「回测报告」
3. 选择报告类型（总体/年度/月度）
4. 点击具体报告查看详情

## 📈 报告示例

### 总体报告示例

```markdown
# 策略回测总体报告

## 测试周期
2016-01-01 至 2026-01-01（10年）

## 核心结果

| 策略 | 信号次数 | 胜率 | 平均收益 | 最优周期 |
|------|---------|------|---------|---------|
| 六脉6红 | 156 | 54.4% | 0.77% | 28天 |
| 买点2 | 89 | 83.3% | 1.11% | 16天 |
| 稳健组合 | 45 | 62.5% | 0.85% | 15天 |
| 激进组合 | 78 | 58.3% | 0.72% | 8天 |

## 最优策略推荐
1. **保守型投资者**：稳健组合（六脉5红+买点2）
2. **积极型投资者**：激进组合（六脉4红+买点2）
3. **趋势跟踪者**：共振组合（多信号确认）
```

## ⚠️ 注意事项

1. **历史数据**：报告基于历史数据，不代表未来表现
2. **市场环境**：不同市场环境下策略表现可能差异较大
3. **风险提示**：投资有风险，报告仅供参考
4. **定期更新**：建议每月更新报告以反映最新数据
